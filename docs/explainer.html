Formalizing Riemann-Roch in Lean 4

A work-in-progress formalization of the Riemann-Roch theorem for algebraic curves. Source on GitHub

Target Theorem

Riemann-Roch
l(D) - l(K - D) = deg(D) + 1 - g
For a divisor D on a smooth projective curve of genus g, with canonical divisor K

Key Results from Cycle 12

  • ell_ge_deg_minus_genus — Lower bound: l(D) ≥ deg(D) + 1 - g

Architecture: FunctionFieldData → FunctionFieldDataWithBound → FunctionFieldDataWithRR. The RR equation is axiomatized (not derived from cohomology), but derived lemmas like l(K) = g use semantic properties (deg_div = 0 for principal divisors).
